# 📚 Documentation Center - User Guide

## Quick Access

The Documentation Center is now built into your app! Here's how to access it:

### Step 1: Open Help
Click the **Help** menu or press the keyboard shortcut (if configured)

### Step 2: Click Documentation Tab
In the Help Dialog, you'll see three tabs:
- **Engine** - Original engine documentation
- **Collisions** - Collision system guide
- **Documentation** ← **NEW!**

### Step 3: Browse & Learn
The Documentation Viewer opens with:
- **Left Sidebar**: Categories and topics
- **Right Content Area**: Selected topic content
